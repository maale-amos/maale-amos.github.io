בצע, וצרף פלט מלא לכל שלב ל-STATUS.md:
  a. git ls-tree origin/master --name-only | grep -E '^index.html$|^data'
     (מה מהישן עדיין בשורש master?)
  b. gh api repos/maale-amos/maale-amos.github.io/pages --jq '{source,build_type}'
  c. gh run list --workflow=build.yml --limit 5
  d. gh run view <last-run-id> --log-failed   (אם נכשל — למה?)
  e. תיקון: על master, מחק מהשורש את index.html הישן + data.json +
     data/ הישנים + כל קובץ root ישן שלא שייך ל-Eleventy. השאר רק:
     src/, .eleventy.js, package.json, package-lock.json, .github/,
     worker/, FACTS.md, CLAUDE.md, NIGHT_TASK.md.
     commit: "phase-2(cleanup): remove legacy root files so Pages
     serves _site build" → push origin master.
  f. ודא build_type=workflow (אם לא: gh api -X PUT .../pages -f build_type=workflow)
  g. gh workflow run build.yml -r master → המתן לסיום (Monitor).
  h. אמת מול ה-URL הציבורי, בלולאה כל 3 דקות עד שעובר:
     curl -s "https://maale-amos.github.io/?nc=$(date +%s%N)" | grep -c "console.cloud"  → חייב 0
     curl -s "https://maale-amos.github.io/?nc=$(date +%s%N)" | grep -c "יוסף שניידר"    → חייב 0
     curl -s "https://maale-amos.github.io/?nc=$(date +%s%N)" | grep -c "166"            → חייב 0
     curl -s "https://maale-amos.github.io/?nc=$(date +%s%N)" | grep -c "איתן סער"       → חייב ≥1
